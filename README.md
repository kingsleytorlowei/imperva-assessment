# Imperva-Assessment

# How to Run 

Start by opening root directory.

## Create Virtual Enviroment
```
python3 -m venv env
```

## Activate Virtual Enviroment
```
source env/bin/activate
```

## Install dependencies
```
pip install -r requirements.txt
```

## Run code
```
python main.py
```

## Run Tests
```
python -m unittest
```
